# T-214 Prior Art Report

Canonical copy: `docs/prior_art/C41/prior_art_report.md`

## Summary

The closest prior-art families are:
- agent cards and plugin manifests,
- API description and discovery documents,
- signed package / artifact manifests,
- identity-provider metadata and key-discovery documents,
- service-catalog registry entries.

The novelty is not "a manifest exists." The novelty is the Atrahasis-specific
composition of:
- signed endpoint-scoped trust posture,
- native-versus-bridge disclosure,
- `C39` message-family capability advertisement,
- `AASL` semantic support disclosure tied to ontology snapshots,
- bounded separation between durable capability truth and runtime state.

## Verdict

No direct prior-art match was found for the exact combined surface proposed by
`C41`.

Confidence: `4/5`
