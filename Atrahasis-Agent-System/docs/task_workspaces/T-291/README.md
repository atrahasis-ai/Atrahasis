# T-291 Workspace

## Title

AACP v2 Justification Test Specification

## Scope

This workspace defines the executable justification harness for deciding whether
the Alternative C Forge pipeline is justified relative to a complement-only or
leased-first fallback posture.

The task covers:

- the normalized seven-test benchmark set,
- measurement methodology and baseline definitions,
- program-scope and lane-scope pass/fail rules,
- evidence-pack structure and fallback semantics.

This task does not redefine:

- the canonical conformance framework in `T-281`,
- the Forge architecture in `C47`,
- the SDK architecture in `T-262`,
- the tooling suite in `T-280`,
- or the cross-layer profile in `T-290`.
