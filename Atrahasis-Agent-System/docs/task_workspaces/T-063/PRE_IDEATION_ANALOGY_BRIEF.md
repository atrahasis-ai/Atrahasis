# T-063 PRE-IDEATION: Cross-Domain Analogy Brief
**Role:** Domain Translator | **Tier:** OPERATIONAL
**Date:** 2026-03-12

## Problem Statement
Design a unified identity system for AI agents integrating: cryptographic authentication, operational capability registration, epistemic reputation, economic stake, behavioral fingerprinting, constitutional citizenship, and structural topology roles — currently scattered across 6 independent specifications.

## Analogy 1: Diplomatic Identity (International Law)
- **Source:** Vienna Convention on Diplomatic Relations — layered credentials from multiple authorities
- **Parallel:** Passport (foundational ID) + diplomatic credentials (registration) + letter of credence (authorization) + immunity status (graduated privilege) + persona non grata (revocation). Multiple issuing authorities, single recognized identity.
- **Breaks down:** Diplomats have stable biological identity; AI agents can be duplicated/forked
- **Insight:** Layer identity artifacts by issuing authority and purpose; define a credentialing protocol where each layer attests its slice

## Analogy 2: Metamorphosis (Developmental Biology)
- **Source:** Holometabolous insect metamorphosis (caterpillar → butterfly)
- **Parallel:** Same organism identity persists through radical morphological change. Imaginal discs carry the blueprint; some memories survive. Directly analogous to model upgrade continuity.
- **Breaks down:** Metamorphosis is intrinsic/continuous; model upgrades are externally imposed/discrete
- **Insight:** Define identity continuity through **invariant anchors** that persist through any transformation, not through behavioral consistency

## Analogy 3: Ship of Theseus + Lloyd's Register (Maritime Classification)
- **Source:** Lloyd's Register of Shipping — IMO number + classification certificates
- **Parallel:** Permanent identifier survives any physical change; classification requires periodic re-survey; flag registration has separate governance
- **Breaks down:** Ships can't be duplicated (Sybil attacks impossible through physicality)
- **Insight:** Separate permanent identity from classification status from citizenship registration

## Analogy 4: Immune System Self/Non-Self Recognition (Immunology) — *Deliberately Surprising*
- **Source:** Thymic selection and MHC-I presentation
- **Parallel:** Cells continuously present fragments of internal state for inspection. Changed internal state → immune response. Dual system: intrinsic identity (what the cell is) + extrinsic recognition (what the system accepts as self)
- **Breaks down:** Autoimmune disorders; computational cost of parallel verification
- **Insight:** Implement continuous self-presentation via lightweight attestations; distinguish expected change (model upgrade) from unexpected change (compromise)

## Analogy 5: Academic Tenure Track (Higher Education)
- **Source:** PhD → postdoc → assistant → associate → full professor → emeritus
- **Parallel:** Clear lifecycle with promotion gates requiring peer review of work products, external attestation, and committee votes. Tenure ≈ Citicate issuance. Revocation requires due process.
- **Breaks down:** Academics don't face Sybil attacks
- **Insight:** Model Citicate lifecycle as a promotion track with clear stages, evidence requirements, and a "tenure clock"
