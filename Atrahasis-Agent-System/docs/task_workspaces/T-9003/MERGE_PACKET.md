# Nanshe Merge Packet

## Core Position

Atrahasis should become a **closed, mission-locked, sovereign intelligence company**.

## Key Claims

- AGI at Atrahasis should mean an internal sovereign recursive intelligence institution, not a public product.
- The strongest capabilities must remain internal.
- External users should receive bounded derivative products, not raw invention access.
- Current `C14`, `C15`, `C18`, and `C22` assumptions are incompatible with this pivot without major redesign.
- Frontier models should be used now through `C44` + `C45` + AACP/AASL discipline, then progressively replaced by native Atrahasis-trained models.

## Non-Negotiables

- no public raw novelty engine
- no unrestricted enterprise invention interface
- no release of Atrahasis-critical self-improvement capabilities
- no founder-above-the-system exception
- no assumption that "private" alone equals "safe"

## Biggest Required Rewrite

`C14` AiBC governance.

The current AiDP/Citicate model assumes a more open constitutional participation structure than a closed AGI institution can tolerate.

## Recommended New Doctrine

1. `Mission Supremacy`
2. `Novelty Embargo`
3. `Bounded Externalization`
4. `Native-First Sovereignty`
5. `Technical Non-Harm Enforcement`

## If Approved, First Canonical Work

1. Mission charter rewrite
2. Closed governance redesign
3. Capability sovereignty membrane
4. Public product tiering
5. Economics and licensing rewrite
6. Model substrate roadmap
7. Repo-wide compatibility audit
