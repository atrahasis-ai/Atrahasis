# Anu Merge Packet

## Core Position

Atrahasis should become a **constitutionally sealed AGI custodian**.

## Key Claims

- `AGI` inside Atrahasis should mean an internal sovereign intelligence asset,
  not a public product surface.
- The outside world should receive bounded projections, licensed technologies,
  and operator-mediated outcomes, not raw novelty access.
- `C47` already points in this direction technically, but `C14`, `C15`, `C18`,
  and `C22` still assume a more open institutional and commercial world.
- External frontier models can be used now as leased cognition behind
  `AACP` / `AASL` / `C44` / `C45`, while Atrahasis builds toward native
  models.

## Non-Negotiables

- no public raw novelty engine
- no unrestricted enterprise invention interface
- no external release of Atrahasis-critical self-improvement primitives
- no founder-above-constitution exception
- no release by contract alone without constitutional review
- no claim that "private" alone makes the system safe

## Distinctive Proposal

Do not think in terms of "open vs closed product."
Think in terms of **four constitutional surfaces**:

1. `Sanctum`
2. `Foundry`
3. `Enterprise`
4. `Public`

Only `Sanctum` should access unrestricted recursive invention.

## Biggest Required Rewrite

`C14` AiBC governance.

The current AiDP / Citicate structure assumes broader constitutional
participation than a sealed AGI custodian can tolerate.

## Recommended New Doctrine

1. `Recursive Custody`
2. `Embargoed Novelty`
3. `Petitioned Externalization`
4. `Constitutional Release Law`
5. `Leased Cognition, Native Sovereignty`
6. `No Moral Theater`

## If Approved, First Canonical Work

1. Mission charter rewrite
2. Internal constitutional stewardship redesign
3. Capability embargo and release-class specification
4. Economics and licensing rewrite
5. Implementation-plan resequencing
6. Spec-by-spec compatibility audit
