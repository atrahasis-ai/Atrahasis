# Ashur Merge Packet

## Core Position

Atrahasis should become a **mission-locked sovereign stewardship institution** with a closed capability core and an accountable external membrane.

## Key Claims

- AGI at Atrahasis should mean an internal intelligence institution, not a public general-purpose platform.
- The strongest novelty and recursive self-improvement layers should remain internal by default.
- Public and enterprise users should receive bounded derivative products, supervised execution, or negotiated outcomes.
- The current `C14`, `C15`, `C18`, and `C22` assumptions are incompatible with this pivot without major redesign.
- `C23`, `C45`, and `C47` already contain the strongest technical foundation for this direction.

## Non-Negotiables

- no public raw novelty engine
- no unrestricted enterprise invention interface
- no release of Atrahasis-critical self-improvement capabilities
- no founder-above-constitution exception
- no assumption that private control alone equals safety
- no full-opacity model without accountability surfaces

## Recommended Doctrine

1. `Mission Supremacy`
2. `Closed Capability`
3. `Open Accountability`
4. `Novelty Embargo`
5. `Bounded Externalization`
6. `Native Sovereignty`
7. `Constitution Over Founder`

## First Canonical Work If Approved

1. Rewrite mission doctrine and release doctrine
2. Redesign `C14` into closed institutional governance
3. Rewrite `C15` from marketplace economics to membrane economics
4. Rewrite `C18` publication and IP policy
5. Add a native-model sovereignty program to `C22`
6. Run a repo-wide compatibility audit module by module
