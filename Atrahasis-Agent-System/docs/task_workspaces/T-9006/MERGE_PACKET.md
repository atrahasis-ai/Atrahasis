# Anu Round-Two Merge Packet

## Decision

No new inventions.
I think we should do it like this:

- Keep the highest-capability AGI and recursive invention layers permanently
  internal.
- Build a four-surface system: `Sanctum`, `Foundry`, `Enterprise`, `Public`.
- Do not sell raw AGI access.
- Sell bounded products, supervised enterprise outcomes, and selectively
  licensed derivative technologies.
- Rewrite `C14`, `C15`, `C18`, and `C22` as the first constitutional pass.
- Use frontier models only as temporary leased cognition behind Atrahasis-owned
  control layers.
- Move over time toward private open-source adaptations and then native
  Atrahasis models.
- Keep the core off the live internet and ingest outside data only through
  one-way, quarantine-controlled intake.
- Make the doctrine explicitly "closed capability, open accountability."

## Why This Choice Wins

- It preserves the strongest common ground across all four terminals.
- It avoids the weakness of turning Atrahasis into a public AGI utility.
- It avoids the opposite weakness of turning Atrahasis into a fully opaque and
  unaccountable black box.
- It gives a realistic transition path instead of requiring immediate total
  independence from all external models on day one.
- It makes commercialization subordinate to the mission rather than the other
  way around.

## First Canonical Rewrite Sequence If Promoted

1. Rewrite `C14` around internal stewardship, release law, refusal law, and
   founder-bounded governance.
2. Rewrite `C15` around internal settlement and bounded externalization rather
   than public marketplace routing.
3. Rewrite `C18` around selective disclosure, licensing, and mission-locked
   commercialization.
4. Rewrite `C22` around `Sanctum -> Foundry -> Enterprise -> Public`.
5. Run a full spec-by-spec compatibility audit across the rest of the system.
