# C39 Simplification Notes

LCML remains bounded because it does not:
- redesign C38's five-layer model,
- define semantic type internals,
- specify handshake/session frames,
- specify transport bindings,
- define tool/resource/prompt business logic beyond message contracts.

It does:
- normalize the current inventory,
- define 19 new classes,
- define the envelope rules those classes require.

This is the minimum message-layer authority Alternative B needs before the downstream protocol specs can proceed safely.
