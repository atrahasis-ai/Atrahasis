# T-211 Prior Art Summary

Canonical report: `docs/prior_art/C39/prior_art_report.md`

High-signal findings:
- A2A and MCP confirm the need for explicit capability families.
- Generic RPC patterns alone are too weak because they do not preserve Atrahasis lineage and semantic-governance constraints.
- The Atrahasis lineage already prefers fixed message classes over ad hoc payload-defined operations.
- The novelty candidate is the bounded family lattice plus lineage and provenance posture, not the existence of request/response classes by itself.
