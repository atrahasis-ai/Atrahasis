# Invention Logs

Each invention gets its own log file: `<INVENTION_ID>.md`

**Owner:** The specialist assigned to the invention.
**Template:** `docs/templates/INVENTION_LOG_TEMPLATE.md`

## Rules
- One file per invention (e.g., `C1.md`, `C2.md`)
- Only the assigned specialist edits their log
- Chronicler may add closure notes
- Commit log updates frequently (small checkpoints)
