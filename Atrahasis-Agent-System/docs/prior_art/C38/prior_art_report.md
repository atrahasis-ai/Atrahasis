# C38 Prior Art Report

Canonical research artifact for C38. Derived from `docs/task_workspaces/T-210/PRIOR_ART_REPORT.md`.

## Summary
- `C4 ASV` supplies the semantics baseline but not sovereign protocol ownership.
- `A2A/MCP` supply transport/tool ecosystems but break the Atrahasis semantic-integrity chain.
- Classic layered network stacks prove the utility of explicit layer contracts.
- Capability security and content-addressed object systems supply security and canonicality primitives.

## Conclusion
The novelty of C38 is not layering itself. It is the Atrahasis-specific composition in which semantic canonicality is defined once, then preserved across messaging, security, session, and transport without lower-layer reinterpretation.
