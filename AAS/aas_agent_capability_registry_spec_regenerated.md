# AAS Agent Capability Registry Specification (Regenerated)

The Agent Capability Registry catalogs the agents used by AAS-RE and
allows the Coordination Kernel to allocate the strongest available team
for each stage of the architecture pipeline.

## Registry Coverage

The registry must include:

- strategic agents
- architecture analysis agents
- code intelligence agents
- governance agents
- simulation agents
- pressure-testing agents
- adversarial-review agents
- council agents
- kernel agents

## Kernel Queries

The Coordination Kernel queries the registry for:

- PAEE swarm staffing
- pressure-test staffing
- adversarial-team staffing
- task graph assignment
- recovery and replay work

## Dynamic Learning

Registry learning must track performance across:

- proposal quality
- council approval
- pressure-test resilience
- adversarial resilience
- task completion

## Objective

The ACR ensures AAS-RE can assign the most capable agents to each phase
of bounded architecture evolution.
